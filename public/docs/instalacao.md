# Instalação e Execução

## 1. Backend (porta 3000)

```bash
git clone https://github.com/brunoogalvani/dashboard-clima-backend
cd dashboard-clima-backend
npm install
npm run dev
```

O backend ficará disponível em:
http://localhost:3000/api/

## 2. Frontend (porta 5173)

```bash
git clone https://github.com/brunoogalvani/dashboard-clima-frontend
cd dashboard-clima-frontend
npm install
npm run dev
```

Acesse o dashboard em:
http://localhost:5173/

## 3. Documentação (docsify)

```bash
npm install -g docsify-cli
docsify serve docs
```

Acesse em:
http://localhost:5173/docs/index.html
