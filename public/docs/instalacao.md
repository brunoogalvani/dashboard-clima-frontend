# Instalação e Execução

## Backend (porta 3000)
```bash
cd backend
npm install
npm run dev
```

## Frontend (porta 5173)
```bash
cd frontend
npm install
npm run dev
```

## Documentação (porta 5173)
```bash
npm install -g docsify-cli
```
Acesse em: [http://localhost:5173/docs/index.html](http://localhost:5173/docs/index.html)
